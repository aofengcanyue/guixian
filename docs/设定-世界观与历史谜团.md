# 设定：世界观与历史谜团

## 文件定位

本文档用于固定《诡仙录》的大世界观、历史谜团、终局真相与长篇伏笔释放节奏。

本文件不等于开篇设定说明。

很多内容属于作者后台设定，不能在第一卷直接讲给读者。

长篇写作原则：

> 世界真相必须分层揭露。第一卷只让读者知道长生宗不对；中期才知道各宗都不对；后期才触碰病天与飞升骗局。

---

# 一、世界基础定位

## 1. 世界表层认知

世人所认知的世界，是一个典型东方仙侠世界。

凡人相信：

- 天地有灵；
- 人可修仙；
- 仙门护世；
- 妖魔作乱；
- 凡人若得仙缘，便可脱离苦海；
- 修士飞升，是天地认可的最高荣耀。

在表层世界中，仙门确实扮演着秩序维护者：

- 镇妖；
- 救灾；
- 赐药；
- 授法；
- 维持城镇与荒野边界；
- 给凡人提供改命可能。

因此，普通凡人不会天然反仙门。

他们崇拜仙门，有现实原因。

---

## 2. 世界暗层真相

暗层世界并不是简单的“仙门都是坏人”。

更准确地说：

> 仙道体系本身建立在一种被长期合理化的病变之上。

修仙者获得力量与寿命，同时逐渐失去作为人的完整性。

宗门为了维持修士稳定、延缓畸变、供养高阶修士，会把凡人、低阶弟子、药童、香火、历史和名字纳入消耗体系。

最恐怖之处在于：

> 这套体系已经运转太久，久到连大多数参与者都以为它就是天经地义的仙道。

---

# 二、世界地理格局

## 1. 青州

第一卷与第二卷前半主要区域。

青州多山、多瘴、多小妖异祸，因此长生宗在此地声望极高。

青州适合承担开篇功能：

- 世界足够封闭，方便悬疑潜伏；
- 长生宗势力可以覆盖多个凡人镇；
- 凡人对仙门依赖强；
- 沈照逃出宗门后仍处于长生宗影响范围内。

重要地点：

- 青石镇；
- 长生宗；
- 柳树村；
- 药契镇；
- 山瘴边界；
- 被废弃的旧祠。

## 2. 人间诸州

中期逐步展开。

每一州可对应不同仙门体系与不同病灶。

建议区域：

| 区域 | 表层特色 | 暗层病灶方向 |
|---|---|---|
| 青州 | 丹道、救疫、延寿 | 以人性炼丹 |
| 剑州 | 剑修、武风、宗门斗法 | 剑意寄生，剑吃主人 |
| 符州 | 符箓、文法、城池秩序 | 文字污染，名字被改写 |
| 香州 | 庙宇、香火、信仰繁盛 | 信徒愿力反噬神像 |
| 血州 | 仙族世家、血脉传承 | 祖血返祖，后代成容器 |
| 荒州 | 魔门、散修、废墟 | 主动承认污染，弱肉强食 |
| 中州 | 仙庭、正史、飞升台 | 仙道秩序核心与历史改写 |

## 3. 仙庭核心区

后期区域。

仙庭不一定是传统天庭，而可以是人间高阶仙门共同维护的秩序中枢。

它负责：

- 编修仙史；
- 审核宗门法统；
- 维护飞升仪式；
- 管理香火与仙籍；
- 给各大仙门的吃人行为提供合法解释。

仙庭不是第一卷内容。

至少第四卷后再逐渐出现其影子。

# 三、世界运行规则

## 1. 仙门护世是真实的

仙门不是纯骗局。

它确实能镇压一部分灾祸。

这点必须保留。

如果仙门毫无价值，只靠欺骗吃人，那么世界逻辑会太浅。

更好的逻辑是：

> 仙门一边护世，一边吃人；一边镇灾，一边制造新的灾；一边救一地凡人，一边把另一批凡人炼成维持秩序的燃料。

这会让沈照的反抗更复杂。

## 2. 妖魔不一定是外敌

表层认知：妖魔是仙门镇压的邪祟。

暗层设定：部分妖魔可能来自：

- 修仙失败者；
- 被剥离名字的人；
- 丹房废料；
- 香火污染；
- 旧史中被抹去的修士；
- 旧时代失败的诡仙或病灶容器。

这能让“仙门镇妖”的行为变得复杂：

他们镇压的也许是自己体系制造出来的后果。

## 3. 灵气有效，但有代价

灵气不是纯粹骗局。

它确实能带来力量。

但长期吸收，会让修士与某种更高层病变机制产生连接。

不同修法只是不同适应方式。

第一卷不要解释“灵气污染”。

前期只写沈照感受到的异常：细针感、哭声、名字模糊、影子慢半拍。

## 4. 名字是一种人间锚点

在这个世界中，名字不只是称呼。

名字代表一个人与人间的连接：

- 家乡；
- 亲人；
- 记忆；
- 愿望；
- 恐惧；
- 未完成的牵挂。

仙门吃人前，往往先弱化名字。

过程可能是：

1. 凡名；
2. 入门名；
3. 法号；
4. 编号；
5. 药材；
6. 空白。

《诡仙录》的核心对抗，就是把空白重新写回名字。

# 四、古史断层

## 1. 正史说法

仙门正史宣称：

远古时，妖魔横行，人间无序。

后来道祖得天授法，开仙道，立宗门，传修行，镇妖魔，分九州，使人间从混乱走向秩序。

正史关键词：

- 道祖授法；
- 仙门立世；
- 妖魔退散；
- 飞升有路；
- 仙庭定史；
- 香火护民。

这套说法不能全假。

它可以有真实部分，但被严重删改。

## 2. 被抹去的旧史

真实旧史可能是：

远古时代并非没有修行，而是存在多条道路。

其中至少包括：

- 以灵气修仙的正仙路；
- 以名字、死愿、残念、旧契为核心的诡录路；
- 以肉身与血脉承载古老力量的血脉路；
- 以凡人工艺、阵法、医术抵御灾祸的无灵路。

后来某次巨大灾变后，正仙路成为主流，并把其他道路定为邪、魔、伪、旁门。

《诡仙录》可能就是旧时代被封禁道路的残留。

## 3. 古史为何断裂

古史断裂原因可以多层叠加：

1. 仙门主动删史；
2. 飞升机制吞掉高阶修士记忆；
3. 病天污染导致历史自我修正；
4. 凡人寿短，无法保存长线真相；
5. 修史官体系将不合仙道叙事的人名抹除。

第五卷《旧史无名》重点揭开这一层。

# 五、病天设定

## 1. 作者后台定义

“病天”不是一个简单怪物。

它更像：

> 一套已经失衡、污染、异化的天道运行机制。

它可能曾经是正常天道。

也可能是某种外来力量寄生天道。

也可能是人间无数修仙者飞升、香火、愿望、恐惧长期堆积后形成的病变规则。

当前建议采用更高级的模糊设定：

> 天不是一个人病了，而是“成仙这件事”本身把天磨病了。

这样能避免终局变成单纯打大怪。

## 2. 病天的表现

病天不会在前期直接显形。

它通过世界规则渗透：

- 灵气让人变强，也让人丢失自己；
- 飞升像荣耀，也像被钓走；
- 香火能护人，也能反向吞噬神像；
- 丹药能救命，也能把别人炼进去；
- 历史能维持秩序，也能抹掉不该存在的人。

病天不是“某处有个怪物”。

病天是：

> 你以为正常的仙道规则，本身就有问题。

## 3. 病天揭露节奏

| 阶段 | 揭露程度 |
|---|---|
| 第一卷 | 不出现“病天”概念，只出现长生宗异常 |
| 第二卷 | 出现“仙缘像契约”的感觉 |
| 第三卷 | 出现“无灵者更干净”的疑问 |
| 第四卷 | 各宗病灶表明问题不是长生宗个案 |
| 第五卷 | 旧史中出现“天疾”“天裂”“旧录”等词 |
| 第六卷 | 飞升异常暗示天上不是净土 |
| 第七卷 | 正式确认天道机制出问题 |
| 第八卷 | 众生立卷，重写局部规则 |

## 4. 病天不应直接人格化

不要让病天像普通反派一样说话、嘲讽、出手。

更好的写法：

- 它通过规则回应；
- 通过雷劫、飞升、香火、仙籍、名字消失来体现；
- 角色越接近真相，越感到整个世界在“纠错”。

这样格局更高。

# 六、飞升骗局

## 1. 表层飞升

世人认为飞升是：

- 修士圆满；
- 天道认可；
- 雷劫洗礼；
- 脱离人间；
- 进入仙界。

飞升者的宗门会立碑、修史、开庆典。

凡人会把飞升者供奉为仙。

## 2. 暗层飞升

后期揭露：

飞升并非进入逍遥仙界。

可设定为：

> 高阶修士在完成病仙路后，被更高层规则收走，成为维持病天机制的一部分。

表现为：

- 飞升者留下空壳或过度干净的遗迹；
- 雷劫像钩子，不像考验；
- 飞升台下有许多断掉的名字；
- 飞升前修士会主动忘记自己是谁；
- 飞升后，人间历史会自动美化其一生。

## 3. 飞升与名字

飞升前，修士必须“脱尽凡名”。

正史说法：

> 凡名为尘，法号为真。

暗层真相：

> 失去凡名，才更容易被飞升机制收编。

沈照后期打断飞升的关键，不是硬抗雷劫，而是让飞升者想起自己的凡名。

这是《诡仙录》与飞升骗局的强关联。

## 4. 第六卷核心场景方向

沈照在第六卷当众打断一次飞升。

他不是简单击败飞升者，而是在飞升台上写出那位修士早已遗忘的凡名、母亲、出身、第一件亏欠之事。

雷劫钩落下时，那位修士短暂醒来，说：

> 原来我不是要上天，我是早就回不了家了。

这一幕可以成为后期大爆点。

# 七、仙庭与仙籍

## 1. 仙庭定位

仙庭不一定是神仙居住的天庭。

它可以是高阶仙门共同维护的秩序机构。

功能：

- 认定正统；
- 编修仙史；
- 管理仙籍；
- 审核宗门；
- 维护飞升台；
- 定义何为魔道。

仙庭的恐怖不在于血腥，而在于：

> 它拥有解释权。

它可以决定：

- 谁是仙；
- 谁是魔；
- 谁的名字可以入史；
- 谁的死亡算功德；
- 哪些灾难应该被记住；
- 哪些人从来没有存在过。

## 2. 仙籍

仙籍是仙庭记录修士身份、功德、法统、飞升资格的核心账本。

表层作用：

- 管理修士；
- 记录功德；
- 确认法统；
- 授予名号。

暗层作用：

- 把修士绑定进仙道系统；
- 把凡俗名字替换为法号；
- 把代价转嫁为功德；
- 把吃人行为美化为护世消耗。

《诡仙录》后期可与仙籍形成对照：

| 仙籍 | 诡仙录 |
|---|---|
| 记录仙人的功德 | 记录被吃掉的人名 |
| 美化秩序 | 显示代价 |
| 抹去不合叙事者 | 补回被抹去者 |
| 让修士脱凡 | 让人记得自己是人 |

# 八、各大仙门病灶方向

## 1. 长生宗：丹病

表层：丹道济世，延寿救疫。

暗层：以人性炼丹，编号替名。

代表主题：

> 药材不可有名。

## 2. 剑宗：剑病

表层：剑心通明，斩妖除魔。

暗层：剑意寄生，剑逐渐成为主人。

病灶表现：

- 剑修越强，越像手中剑；
- 剑会替主人选择敌人；
- 断剑中残留历代剑主名字；
- 剑宗高阶者不是人在御剑，而是剑在用人。

代表主题：

> 人若只剩锋芒，是否还算人？

## 3. 符宗：字病

表层：符通天地，定法安民。

暗层：文字污染，名字被法度改写。

病灶表现：

- 符文能改人身份；
- 城中户籍会自动修正错误；
- 被判为妖者，身体逐渐长出妖相；
- 写错一个人的名字，就能改变他的命运。

代表主题：

> 谁掌握文字，谁定义人。

## 4. 香火道：愿病

表层：护佑万民，神像显灵。

暗层：信徒愿望反向吞噬神像与修士。

病灶表现：

- 神像眼中有无数愿望；
- 求子、求寿、求财、求仇混成污浊香火；
- 被供奉者越来越不像自己；
- 信徒越虔诚，神像越饥饿。

代表主题：

> 众愿不一定是善。

## 5. 仙族世家：血病

表层：血脉高贵，祖荫庇佑。

暗层：祖血返祖，后代成为旧祖容器。

病灶表现：

- 越纯的血脉越容易被祖先意志占据；
- 家族族谱会吞掉不合格后代；
- 新生儿出生时带有祖辈记忆；
- 家主不是一个人，而是一条血脉在说话。

代表主题：

> 血脉传承，究竟传的是人，还是枷锁？

## 6. 魔门：认病者

表层：邪法乱世，不守仙规。

暗层：魔门不一定更善，但他们更诚实地承认污染。

病灶表现：

- 主动利用畸变；
- 不抹去代价；
- 强者吞弱者；
- 不披慈悲画皮。

代表主题：

> 承认自己吃人，不代表不吃人。

魔门可以反衬仙门，但不能洗白成好人阵营。

# 九、历史谜团清单

以下谜团用于长篇持续追读。

## 第一层谜团：长生宗局部谜团

适用：第一卷至第二卷

1. 开灵丹中的“三十七目”是什么？
2. 长生宗为何要抹去药童名字？
3. 陆怀真当年救的三十七童是否还有幸存者？
4. 清虚真人是否早就知道沈照异常？
5. 青石镇当年瘟疫是否与长生宗有关？
6. 沈照的祖传残页为何会在青石镇？

## 第二层谜团：仙门体系谜团

适用：第二卷至第四卷

1. 为什么凡人与仙门之间存在药契？
2. 无灵根者为何反而更干净？
3. 各宗修法为何都对应某种病灶？
4. 妖魔与修仙失败者是否有关？
5. 长生宗是否只是更大体系的一个样本？

## 第三层谜团：历史谜团

适用：第五卷

1. 仙门正史为什么没有被吃掉的人？
2. 旧时代是否存在不依赖灵气的道路？
3. 《诡仙录》是否曾有完整版本？
4. 修史官体系究竟抹去了多少人？
5. 道祖开仙道之前，人间如何抵御灾祸？

## 第四层谜团：飞升谜团

适用：第六卷

1. 飞升者到底去了哪里？
2. 雷劫为什么像钩子？
3. 飞升前为何必须断凡名？
4. 飞升台下的断名是谁留下的？
5. 仙界是否真实存在？

## 第五层谜团：病天谜团

适用：第七、八卷

1. 天道为何会病？
2. 是谁最早发现天病？
3. 正仙路是否一开始就是错的？
4. 诡仙录为何能对抗仙籍？
5. 沈照最终是治天、换天、断仙，还是让人间不再依赖天？

# 十、伏笔释放节奏

## 第一卷：只露局部异常

允许出现：

- 开灵丹有眼；
- 名字被抹；
- 早课、灵粥、感恩录异常；
- 静养院与养药院；
- 陆怀真旧名册；
- 阿灯无灵根更干净；
- 丹房吃人局部真相。

禁止出现：

- 病天概念；
- 飞升骗局；
- 仙庭深层设定；
- 各宗病谱全貌；
- 《诡仙录》真正来历。

## 第二卷：药契与凡人依赖

允许出现：

- 凡人与仙门的契约；
- 凡人不愿被救；
- 沈照发现没有仙门后人间会失序；
- 长生宗影响扩展到凡间。

## 第三卷：无灵之地

允许出现：

- 无灵者更干净；
- 凡人技术与仙道冲突；
- 剥病雏形；
- 第一个低污染生存区。

## 第四卷：诸宗病谱

允许出现：

- 各宗病灶；
- 魔门反衬；
- 沈照从记录个体转向记录体系。

## 第五卷：旧史无名

允许出现：

- 修史官；
- 仙史造假；
- 旧时代多道路；
- 《诡仙录》残缺来源线索。

## 第六卷：飞升之钩

允许出现：

# 十一、世界观写作原则

## 1. 先写具体人，再写大真相

不要从“天道病了”开讲。

要从“阿灯不想被炼成丹”开始。

世界观必须通过具体人物和具体事件释放。

## 2. 每个真相都要改变主角处境

真相不是设定展示。

每次揭露都应该让沈照更难：

- 知道长生宗吃人，所以他必须逃；
- 知道凡人依赖仙门，所以他不能只破坏；
- 知道各宗都有病，所以他不能只复仇；
- 知道历史被改写，所以他必须找记录权；
- 知道飞升是骗局，所以他必须面对整个仙道终极目标。

## 3. 世界不能只有黑暗

必须保留人间烟火：

- 糖画；
- 灯会；
- 祠堂；
- 农田；
- 市集；
- 母亲给孩子缝衣；
- 凡人努力活着。

只有人间值得留恋，沈照的记录才有意义。

## 4. 仙门不能全员脸谱化

许多低阶修士并不知道真相。

许多仙门弟子是真心想镇妖救人。

有些修士本身也是体系受害者。

这样世界才厚。

## 5. 终局不是简单杀光仙人

最终目标不是清算所有修士，而是让仙道代价无法继续隐藏。

沈照要建立的是一种新规则：

> 任何以人为代价的成仙，都必须显出那个人的名字。

---

# 十二、终局方向预案

终局可采用“苍凉立道”路线。

沈照不登仙帝之位。

他也不获得逍遥长生。

他最终将自己写入《诡仙录》最后一页，成为维持新规则的执卷者或活书。

终局新规方向：

- 炼一人丹，丹上显一人名；
- 杀一凡人，仙籍裂一笔；
- 抹一段历史，《诡仙录》补一页；
- 飞升者必须背负完整因果；
- 凡人可以选择不成仙，也有活路。

最终主题落点：

> 人可以死，但不能被无名吞食。

---

# 十三、当前定稿结论

《诡仙录》的世界观，不是简单“正道虚伪、主角复仇”。

它真正的长篇张力是：

> 一个曾经有效、曾经护世、曾经被众生相信的仙道体系，已经在漫长岁月中病变成吃人的秩序。沈照不是从一开始就要毁掉世界，他只是从救下阿灯开始，一点点发现：若不重写规则，人间永远只能把自己的孩子献给仙缘。

因此，所有历史谜团、仙门病灶、飞升骗局、病天真相，最终都要回到同一个问题：

> 人间是否还有不靠吃人也能活下去的路？
