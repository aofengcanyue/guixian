## 一、势力总览

诡仙世界的势力格局，是“天庭崩塌”后“叙事权力”重新分配的结果。四大正统势力（玄门、魔宗、妖庭、佛门）各自“曲解”历史，建立自己的“教义”，在表界和里界“割据”。三大隐脉（守墓人、换命师、无名者）在“暗处”活动，知晓“部分”真相。批注者则“超越”势力，是“读者意识”的具象化。

| 势力 | 类型 | 核心地点 | 领袖 | 第四纪元状态 |
|------|------|----------|------|--------------|
| 玄门 | 四大正统 | 青州（表界）、酆都废墟边缘（里界） | 镜婆婆（旧）、李忘尘（新） | 分裂为“新玄门”和“旧玄门” |
| 魔宗 | 四大正统 | 里界魔域（位置不固定） | 天魔（意识体，后消散） | 解体，魔修散落四方 |
| 妖庭 | 四大正统 | 归墟海边缘 | 龙魂（后沉睡）、白媚（代理） | 分裂，部分归顺忘川 |
| 佛门 | 四大正统 | 表界（原址）、里界（念祟聚集地） | 苦行僧（残余） | “集体飞升”（实为被诡异同化） |
| 守墓人 | 三大隐脉 | 真君墓（位置未知） | 七人议会 | 仍存在，成员更替数代 |
| 换命师 | 三大隐脉 | 落星镇（表界） | 换命师沈渊（后转化） | 解散，首领成为图书管理员 |

## 二、四大正统

### 2.1 玄门

#### 2.1.1 总览

| 项目 | 内容 |
|------|------|
| 全称 | 玄门（原“无情道”） |
| 核心传承 | 《太上忘情录》 |
| 修炼特色 | 以“绝情”压制诡异侵蚀，追求“无我” |
| 暗面 | 高层早已不是人类，是“绝情”这个概念成精 |
| 第四纪元状态 | 分裂为“新玄门”（改革派）和“旧玄门”（保守派） |
| 核心地点 | 青州分舵（表界）、酆都废墟边缘总坛（里界） |

#### 2.1.2 起源与历史

玄门由“绝情真君”在天庭纪元创建，原名“无情道”。绝情真君认为“情感”是叙事的“污染源”，只有“无情”才能“清醒”看到真相。

- **天庭纪元**：无情道镇压“情欲之渊”，弟子稀少但精锐。
- **崩塌纪元**：绝情真君未参与“改写计划”，幸存。她“隐居”在玄门深处，成为“概念成精”——不再是“人”，而是“绝情”本身的具象化。
- **诡道纪（早期）**：玄门成为四大正统之首，主张“绝情修仙”，弟子众多。
- **诡道纪（中期）**：玄门内部出现“分歧”——部分弟子认为“绝情”不是“唯一”出路，开始“秘密”研究“情感”与“叙事”的关系。
- **诡道纪（当下·沈渊时代）**：玄门正式“分裂”为“新玄门”和“旧玄门”。

#### 2.1.3 绝情真君

- **原名**：无情（女）
- **大道**：绝情
- **性格**：冷漠，但“渴望”被理解（矛盾）。
- **状态**：幸存，成为“概念成精”——不再是“人”，而是“绝情”的具象化。她的“遗蜕”在旧玄门禁地（冰窟）中，可以“对话”。
- **与沈渊的互动**：沈渊在旧玄门禁地“见到”她的遗蜕，获得“成为真君而不失自我”的“方法”——将“自我”转移到“纸扎城”。

#### 2.1.4 修炼体系

**核心功法**：《太上忘情录》

| 境界 | 对应“绝情”层次 | 畸变 |
|------|----------------|------|
| 见诡 | 开始“压制”情感 | 情感“淡薄”，但“偶尔”失控 |
| 饲诡 | 伴生诡“无情鬼”（吸收情感） | 情感“消失”，但“记忆”保留 |
| 化诡 | 诡相“冰雕”（冷漠、坚硬） | 无法“理解”他人的痛苦 |
| 掌诡 | 诡域“冰封”（冻结情感、叙事） | 域内“无”情感，但域主“孤独” |
| 炼诡 | 诡器“忘情碑”（封印情感） | 失去“所有”情感，变成“规则” |
| 诡仙 | 棺中“冰冻”沉睡 | 梦中“无”情感，只有“因果” |
| 劫仙 | 三灾“情感灾”（替代“神魂灾”） | 情感“分裂”，可能“爆发” |
| 真君 | 与“绝情”大道同化 | 成为“绝情”本身 |

#### 2.1.5 组织架构

**旧玄门（保守派）**：
- **领袖**：镜婆婆（劫仙，后“转化”）
- **核心成员**：数十名弟子（化诡境、掌诡境）
- **特点**：坚持“绝情”，排斥“情感”，认为“新玄门”是“叛徒”。
- **结局**：镜婆婆被沈渊“转化”后，旧玄门“解散”，弟子“选择”——愿意“接纳情感”的去新玄门，不愿意的“离开”。

**新玄门（改革派）**：
- **领袖**：李忘尘（诡仙，男，约300岁）
- **核心成员**：数百名弟子（见诡到劫仙不等）
- **特点**：主张“与诡异共存”，接纳“情感”，研究“叙事”而非“绝情”。
- **与沈渊的关系**：合作，支持沈渊对抗“大劫”，后“并入”忘川联盟。

**玄门青州分舵**：
- **位置**：青州山区，山腹中
- **分舵主**：镜婆婆（兼）
- **特点**：负责“选拔”见诡者作为“耗材”（修补剧情黑洞）。
- **结局**：沈渊逃离后，分舵被新玄门“接管”。

#### 2.1.6 重要地点

**旧玄门禁地（冰窟）**：
- 位置：酆都废墟边缘
- 描述：一座冰窟，温度极低，绝情真君的“遗蜕”躺在冰棺中。
- 规则：进入者会“情感”逐渐“冻结”，必须“无情”才能“存活”。
- 沈渊的进入：他用“纸人”身份（情感本就是“画”的）抵抗“冻结”。

**新玄门总部**：
- 位置：青州（表界），一座“悬浮”的山峰
- 描述：建筑“现代”（相对于旧玄门），弟子“活跃”，气氛“开放”。
- 特点：研究“叙事”，与散修联盟“合作”。

#### 2.1.7 玄门的“暗面”

- **“耗材”计划**：旧玄门用低阶弟子的“死亡”填补剧情黑洞，维持叙事“稳定”。弟子们被“欺骗”，以为“任务”是“正常”的。
- **镜婆婆的“真相”**：她不是“人”，而是“镜子”概念成精。她的“身体”是镜面，“反射”一切攻击，但“无法创造”。
- **绝情真君的“困境”**：她“渴望”被理解，但“绝情”大道让她“无法”表达。她“选择”沉睡在冰窟，等待“有人”来“理解”她（沈渊“部分”做到了）。

#### 2.1.8 沈渊与玄门的关系

- **初期**：沈渊加入玄门外围，发现“耗材”计划，逃离。
- **中期**：沈渊被玄门“追杀”（镜婆婆），但“反杀”。
- **后期**：沈渊与“新玄门”合作，对抗“旧玄门”和“大劫”。
- **结局**：旧玄门“解散”，新玄门“并入”忘川联盟。


### 2.2 魔宗

#### 2.2.1 总览

| 项目 | 内容 |
|------|------|
| 全称 | 魔宗（原“天魔道”） |
| 核心传承 | 《他化自在天魔功》 |
| 修炼特色 | 主动拥抱畸变，以疯狂对抗疯狂 |
| 暗面 | 宗主就是“天魔”，所有弟子都是它的分身候选 |
| 第四纪元状态 | 解体，魔修散落四方 |
| 核心地点 | 里界魔域（位置不固定，随“天魔”意识移动） |

#### 2.2.2 起源与历史

魔宗由“天魔”（非真君，而是“原初存在”的“噩梦”碎片）在天庭纪元末期创建。天魔“诱惑”修士“拥抱”畸变，以“疯狂”对抗“叙事的束缚”。

- **天庭纪元末期**：天魔“出现”（从渊界“渗透”），开始“诱惑”对天庭不满的修士。
- **崩塌纪元**：天魔“趁乱”扩大势力，吸收“被天道抛弃”的修士。
- **诡道纪（早期）**：魔宗成为四大正统之一，与玄门“对立”，主张“疯狂修仙”。
- **诡道纪（中期）**：魔宗内部“分裂”——部分魔修“渴望”秩序（后成为“新魔宗”？未设定），部分“坚持”疯狂。
- **诡道纪（当下·沈渊时代）**：魔宗“衰落”，被忘川“击败”，宗主“天魔”消散，魔宗“解体”。

#### 2.2.3 天魔

- **本质**：不是“修士”，而是“原初存在的噩梦碎片”——“所有”恶意的“集合体”。
- **形态**：没有“实体”，只有“意识”（一团“黑色”的光）。
- **能力**：
  - “天魔乱”：唤醒修士的“心魔”，让修士“自相残杀”。
  - “分身候选”：所有魔修都是天魔的“潜在”身体，修炼越“成功”，越接近“成为”天魔的“新身体”。
- **结局**：沈渊用“叙事编织”将“恶意”与“善意”平衡，天魔“失去”恶意支撑，“消散”为碎片，融入里界。

#### 2.2.4 修炼体系

**核心功法**：《他化自在天魔功》

| 境界 | 对应“疯狂”层次 | 畸变 |
|------|----------------|------|
| 见诡 | 主动“拥抱”恐惧 | 虹膜“血红”，夜“狂欢”而非“不能寐” |
| 饲诡 | 伴生诡“心魔”（放大欲望） | 欲望“失控”，可能“自残” |
| 化诡 | 诡相“魔相”（扭曲、恐怖） | 失去“理智”，但“获得”疯狂的力量 |
| 掌诡 | 诡域“魔域”（混乱、杀戮） | 域内“规则”崩溃，但域主“享受”混乱 |
| 炼诡 | 诡器“魔器”（吞噬生命） | 失去“人性”，变成“半魔” |
| 诡仙 | 棺中“血棺”沉睡 | 梦中“杀戮”，可能“被”天魔“取代” |
| 劫仙 | 三灾“疯狂灾”（替代“神魂灾”） | 完全“疯狂”，成为“天魔”的“分身” |
| 真君 | 成为“天魔”本身（非“大道”） | 失去“自我”，成为“恶意”的容器 |

#### 2.2.5 组织架构

**魔宗（鼎盛时期）**：
- **宗主**：天魔（意识体）
- **魔子**：七名掌诡境修士（血、骨、魂、影、毒、咒、灾）
- **弟子**：数千名魔修（见诡到劫仙不等）
- **特点**：没有“固定”总部，随“天魔”意识“移动”；内部“竞争”激烈，强者“吞噬”弱者。

**魔宗（沈渊时代）**：
- **衰落**：沈渊击败“血魔子”、“影魔子”等，魔宗实力“大减”。
- **解体**：沈渊“平衡”天魔的“恶意”，天魔“消散”，魔宗“解体”。
- **后续**：魔修“散落”四方——有的“加入”忘川（被“净化”），有的“隐居”，有的“继续”作恶（但力量“大减”）。

#### 2.2.6 重要人物

**厉血**（化诡境）：
- 角色：魔宗使者，试图“夺取”纸扎城。
- 结局：被沈渊击败，重伤“逃走”（后未再出现）。

**血屠**（掌诡境）：
- 角色：魔宗长老，带领魔修“总攻”纸扎城。
- 能力：诡域“血海”（淹没一切，污染信念）。
- 结局：被纸鸢“锁链”束缚，自尽（用血祭自爆）。

**骨婆婆**（掌诡境）：
- 角色：魔宗长老，传授“魔宗炼诡术”（高效但“不稳定”）。
- 能力：炼制“骨器”。
- 阴谋：试图“偷取”纸扎城的“升格秘密”。
- 结局：被沈渊的“叙事陷阱”捕获，变成“骨塔”（纸扎城北门防御）。

**七魔子**：

- 骨魔子：（未出场，可能“隐居”）
- 魂魔子：（未出场）
- 影魔子：被纸鸢击败
- 毒魔子：（未出场）

#### 2.2.7 魔宗的“暗面”

- **“分身候选”真相**：所有魔修都是天魔的“潜在”身体。修炼《他化自在天魔功》“成功”，意味着“自我”逐渐被“天魔”取代，最终“成为”天魔的“新身体”。
- **“血祭”本质**：魔修通过“血祭”转移“污染”，但“因果污染”会让他们“成为”更大的“反派”。
- **“疯狂”的代价**：魔修“拥抱”疯狂，获得“短期”力量，但“长期”会“失去”理智，变成“行尸走肉”。

#### 2.2.8 沈渊与魔宗的关系

- **初期**：魔宗试图“夺取”纸扎城，沈渊“抵抗”。
- **中期**：魔宗“合作”（传授炼诡术），但暗中“偷取”秘密。
- **后期**：沈渊“击败”魔宗主力，天魔“消散”。
- **结局**：魔宗“解体”，部分魔修“归顺”忘川。


### 2.3 妖庭

#### 2.3.1 总览

| 项目 | 内容 |
|------|------|
| 全称 | 妖庭 |
| 核心传承 | 《万灵归元诀》 |
| 修炼特色 | 与“上古妖诡”签订血契，半人半妖 |
| 暗面 | 妖庭深处供奉着一具“龙尸”，但龙尸每隔百年会“眨眼” |
| 第四纪元状态 | 分裂，部分归顺忘川 |
| 核心地点 | 妖庭（归墟海边缘，一座悬浮的宫殿） |

#### 2.3.2 起源与历史

妖庭由“上古妖诡”（非真君，而是“原初存在”梦境中的“兽”）在天庭纪元创建。妖庭的“核心”是“龙尸”——一具“死去”但“未完全死”的巨龙。

- **天庭纪元**：妖庭“臣服”于天庭，真君们“利用”妖族“镇压”里界。
- **崩塌纪元**：妖庭“中立”，未参与“改写计划”。龙尸“第一次”眨眼。
- **诡道纪（早期）**：妖庭成为四大正统之一，主张“半人半妖”修仙。
- **诡道纪（中期）**：妖庭内部“分裂”——部分妖族“渴望”自由（白媚带领），部分“坚持”侍奉龙魂。
- **诡道纪（当下·沈渊时代）**：龙魂“苏醒”，妖庭“进攻”纸扎城，被沈渊“击退”。龙魂“沉睡”，妖庭“分裂”。

#### 2.3.3 龙尸与龙魂

**龙尸**：
- 描述：一具“巨大”的巨龙尸体，躺在妖庭深处。尸体“不腐”，每隔百年“眨眼”。
- 本质：龙魂的“容器”。龙魂“沉睡”在龙尸中，“眨眼”是龙魂“苏醒”的前兆。

**龙魂**：
- 本质：一条“被主人抛弃”的龙，“执念”是“复活”和“复仇”。
- 真名：“被主人抛弃的龙”。
- 能力：
  - 龙息：“叙事之火”，可以“燃烧”记忆、情感、存在。
  - 龙威：压制“自我意识”。
- 结局：沈渊用“真名”驾驭龙魂，命令它“继续沉睡”。龙魂“服从”，回到龙尸中“沉睡”。

#### 2.3.4 修炼体系

**核心功法**：《万灵归元诀》

| 境界 | 对应“妖化”层次 | 畸变 |
|------|----------------|------|
| 见诡 | 与“妖诡”签订血契 | 身体“部分”妖化（如狐耳、龙鳞） |
| 饲诡 | 伴生诡“妖魂”（妖诡的碎片） | 获得“妖诡”的能力，但“妖诡”可能“反噬” |
| 化诡 | 诡相“妖兽”形态 | 完全“妖化”，但“理智”保留 |
| 掌诡 | 诡域“妖域”（兽性、野性） | 域内“人类”变弱，“妖族”变强 |
| 炼诡 | 诡器“妖器”（如龙鳞、狐尾） | 失去“人性”，更接近“妖” |
| 诡仙 | 棺中“兽棺”沉睡 | 梦中“成为”妖兽，可能“永远”无法“变回”人 |
| 劫仙 | 三灾“兽性灾”（替代“肉身灾”） | 身体“完全”妖化，但“意识”保留 |
| 真君 | 成为“妖圣”（与“兽性”大道同化） | 失去“人类”身份，成为“妖兽”的“神” |

#### 2.3.5 组织架构

**妖庭（鼎盛时期）**：
- **最高存在**：龙魂（沉睡，但“偶尔”苏醒）
- **代理领袖**：白媚（狐妖，掌诡境）

- **中级批注者**：能写“分析”和“猜测”，有“一定概率”让猜测“成真”。

**两人的关系**：
- 对立，但“目的”相同——让故事“更精彩”。
- 在第九卷，两人“代表”批注者的“集体意识”，与沈渊“对话”。

### 4.4 批注者与沈渊的关系

- **第一卷**：批注“出现”，沈渊“看到”。
- **第三卷**：批注“开始”与沈渊“互动”（沈渊“尝试”回复）。
