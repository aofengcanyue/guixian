# 《诡仙录》人物设定文档

> 版本：最终定稿  
> 类型：克苏鲁式恐怖 + 东方诡仙 + 强Meta叙事 + 存在主义哲学  
> 核心精神：人以身为饵，饲诡登阶。力量即污染，改写即代价，故事从未属于你。


## 一、主角：沈渊

### 1.1 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 沈渊 |
| 别名 | 纸人沈渊、忘川阁主、叙事真君（半步）、叙事管理员 |
| 年龄 | 出场约二十余岁，故事跨越约五百余年（含沉睡） |
| 身份 | 忘川书肆掌柜 → 玄门外围弟子 → 忘川阁主 → 纸扎城域主 → 诡仙 → 劫仙 → 叙事真君（半步） → 叙事管理员 |
| 境界 | 见诡 → 饲诡 → 化诡 → 掌诡 → 炼诡 → 诡仙 → 劫仙 → 真君（半步） |
| 种族 | 凡人 → 纸人（第四卷后） |
| 特色 | 贪生怕死但选择勇敢，有“叙事干涉”天赋，多重自我共存 |

### 1.2 外貌与性格

**外貌**：
- 初始：普通青年，面容温和，右手掌心有“缝”（后睁开为眼睛）。
- 纸人化后：身体半透明，皮肤有纸张纹理，右手眼睛睁开。左半身曾透明后被重绘。
- 真君后：身体近乎透明（介于存在与不存在之间）。

**性格**：
- **表层**：懦弱、贪生怕死、喜欢给话本改“大团圆结局”。
- **深层**：对“结局”有病态执着——无法忍受“故事就这样结束”。善良但不圣母，愿意为“在意的人”牺牲，但也会谨慎计算得失。
- **成长**：从“逃避命运”到“试图改写命运”再到“接受自己就是命运的一部分，但选择在空白处写批注”。

**口头禅/内心独白**：
- “万一能改呢？”
- “我是纸人。纸人不会怕。”（后变成“纸人也可以哭”）
- “我不记得了。但应该是对的。”

### 1.3 能力体系

#### 天赋：叙事干涉
- **描述**：沈渊与“叙事大道”有天然亲和，能“影响概率”，让事情按“更有故事性”的方向发展。
- **双刃剑**：“故事性”往往意味着悲剧、反转、失去。
- **表现**：他总能遇到“奇遇”，但也总付出“代价”。

#### 核心能力（按境界）

| 境界 | 能力 | 描述 |
|------|------|------|
| 见诡 | 灵视（初级） | 看到黑线（命运/因果线） |
| 饲诡 | 失败预演 | 腹中旧人在脑中预演失败可能 |
| 化诡 | 多重自我 | 可调用梦境中其他沈渊的记忆和能力 |
| 掌诡 | 域主权限 | 在纸扎城内利用“谎言即真实”规则 |
| 炼诡 | 诡器使用 | 醒世钟、伪本、记忆之梭 |
| 诡仙 | 叙事深层访问 | 棺中沉睡经历其他纪元 |
| 劫仙 | 叙事共感 | 可感知他人叙事线，帮助选择分支 |
| 真君 | 叙事编织 | 将不同时间线/分支编织在一起 |

#### 主要诡器

| 诡器 | 原料记忆 | 能力 | 使用代价 |
|------|----------|------|----------|
| 醒世钟 | 母亲 | 敲响让周围“清醒”看到黑线 | 每次失去一段记忆 |
| 伪本 | 弟弟沈墨 | 小范围临时改写现实 | 失去一段因果 |
| 记忆之梭 | 忘川书肆 | 进入他人梦境 | 可能迷失在梦中 |

### 1.4 人物弧线

**第一阶段（卷1-2）：觉醒与逃避**
- 从凡人到修士，被迫走上诡仙之路。
- 核心冲突：怕死 vs 不得不面对死亡。
- 转折：第一次“改写”（杀画皮仙），付出代价（失去弟弟记忆）。

**第二阶段（卷3-4）：接纳与蜕变**
- 接受“多重自我”，成为纸人。
- 核心冲突：保留自我 vs 融合他人。
- 转折：选择“梦境融合”而非吞噬其他沈渊；选择“成为纸人”而非逃避清明雨。

**第三阶段（卷5-6）：失去与孤独**
- 炼制诡器，失去母亲、弟弟、忘川书肆的记忆；棺中沉睡五百年，物是人非。
- 核心冲突：变强 vs 失去自我。
- 转折：拒绝换命师交易；苏醒后接受“孤独”，但仍选择“继续”。

**第四阶段（卷7-8）：共存与超越**
- 与三灾共存，与叙事大道融合（保留自我）。
- 核心冲突：成为真君 vs 保留自我。
- 转折：腹中旧人“牺牲”换融合；与真君沈渊、换命师沈渊和解。

**第五阶段（卷9）：选择与守望**
- 拒绝成为“执笔人”，选择“叙事管理员”。
- 核心冲突：自由 vs 责任。
- 结局：与纸鸢共治渊界，投影在纸扎城“生活”，故事“继续”。

| 真君沈渊 | 另一种可能性 | 最终和解，成为“守护支线”的盟友 |
| 换命师沈渊 | 另一种可能性 | 最终和解，成为“图书管理员” |

### 1.6 代价清单（部分）

| 失去的内容 | 原因 | 卷 |
|------------|------|-----|
| 弟弟沈墨的记忆 | 第一次改写（杀画皮仙） | 1 |

| 与陈舟最后一次对话的记忆 | 叙事编织 | 8 |
| 恐惧的情感 | 多次代价叠加 | 全卷 |

### 1.7 结局状态

- **本体**：留在渊界，与纸鸢本体共为“叙事管理员”，维护原初存在的梦境。
- **投影**：在纸扎城与纸鸢投影“生活”，看“永恒黄昏”。
- **自我认知**：“我是沈渊。我是纸人、真君、叙事管理员、纸鸢的伴侣。我不记得母亲、弟弟、书肆的样子。但我记得‘纸鸢’这个名字。这就够了。”


## 二、女主：纸鸢

### 2.1 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 纸鸢 |
| 别名 | 纸鸢投影（分身）、叙事管理员（本体） |
| 身份 | 纸扎城第一个纸人 → 修士 → 真人 → 叙事管理员 |
| 境界 | 纸人（无境界）→ 见诡 → 饲诡 → 化诡 → 叙事管理员（特殊） |
| 种族 | 纸人 → 真人（第四卷） → 叙事管理员（特殊存在） |
| 特色 | 唯一知道自己是纸人的纸人，渴望“真正活一次”或“真正死一次” |

### 2.2 外貌与性格

**外貌**：
- 纸人形态：纸折的女性形象，有墨迹画出的五官。身体会随“信念”变化（半透明、变薄等）。
- 真人形态：有血有肉，但保留了纸张的某些特征（如皮肤极白）。
- 叙事管理员形态：化为“光”，中心有“纸鸢”（风筝）形状。

**性格**：
- **表层**：温柔、坚韧、有点忧郁（因为知道自己是假的）。
- **深层**：渴望“真实”——不是“变成人”，而是“拥有选择的权利”。善良但不软弱，愿意为“在意的人”牺牲。
- **成长**：从“迷茫的纸人”到“主动寻找真实的修士”到“接受‘共存’也是真实”到“成为叙事管理员，守护他人的故事”。

**口头禅**：
- “你是假的。但假的也可以是真的。”
- “我等你。在‘永恒’里。”
- “自由不是‘什么都能做’，而是‘可以自己选择怎么面对不能做的事’。”

### 2.3 能力体系

#### 纸人时期（纸扎城内）
- **信念之力**：通过“相信”获得临时能力（如飞行）。
- **文曲笔（借用）**：可“画”出攻击或防御。
- **叙事感知（初级）**：作为文曲真君的“女儿”，对叙事敏感。

#### 修士时期（真人）
- **诡相**：纸鸢（风筝）——断线，但自己选择方向。
- **文曲笔（伴生诡）**：笔与纸鸢融合，可在胸腔内“书写”信念。
- **情绪感知**：作为纸人时习得的“共情”能力。

#### 叙事管理员时期
- **叙事维护**：维护渊界叙事的稳定，修补漏洞。
- **投影分身**：可将意识投影到纸扎城。
- **叙事沟通**：与沈渊（叙事管理员）实时联系。

### 2.4 人物弧线

**第一阶段（卷4）：觉醒**
- 在纸扎城作为普通纸人卖纸花，但内心有“疑惑”。
- 遇到沈渊，得知自己是被文曲真君创造的“第一个纸人”。
- 决定修炼，成为“真人”。

**第二阶段（卷4中）：转化**
- 完成“见诡”仪式，诡相为“纸鸢”。
- 豢养文曲笔为伴生诡，胸腔内壁被写上“我是一个人”。
- 代价：身体变薄、透明化。

**第三阶段（卷4末）：成为真人**
- 用“真人之血”、“龙鳞”等材料，在文曲塔进行转化仪式。
- 成功变成“真人”，但失去大部分记忆（包括与沈渊的过去）。
- 重新“喜欢”上沈渊（因为他的温柔）。

**第四阶段（卷4末-卷5）：成为叙事管理员**
- 为了“保护”沈渊和纸扎城，主动“代替”沈渊留在渊界。
- 与原初存在碎片融合，成为“叙事管理员”。
- 代价：失去“自我”（但保留了对沈渊的记忆作为“锚点”）。

**第五阶段（卷6-9）：共存与回归**
- 在渊界维护叙事，与沈渊“共治”。
- 沈渊成为“半步真君”后，纸鸢可以从渊界“投影”到纸扎城。
- 最终，纸鸢投影与纸鸢本体“融合”，成为“完整”的纸鸢（本体力量+投影记忆）。
- 与沈渊“共治”渊界，投影在纸扎城“生活”。

### 2.5 关键关系

| 角色 | 关系 | 影响 |
|------|------|------|
| 沈渊 | 伴侣/锚点 | 他是她“成为真人”的动力，也是她“留在渊界”的理由 |
| 文曲真君 | 创造者/父亲 | 她继承了他的“自由”执念 |
| 纸人们 | 同类/子民 | 她是纸人们的“象征”和“守护者” |
| 纸鸢投影 | 分身/另一个自己 | 最终融合，成为“完整” |

### 2.6 结局状态

- **本体**：与沈渊本体共为“叙事管理员”，在渊界维护原初存在的梦境。
- **投影**：与沈渊投影在纸扎城“生活”，看“永恒黄昏”。
- **自我认知**：“我是纸鸢。我是纸人、真人、叙事管理员、沈渊的伴侣。我‘记得’他。这就‘够’了。”


## 三、反派/镜像：未来的沈渊（三个版本）

### 3.1 真君沈渊（融合派）

| 项目 | 内容 |
|------|------|
| 身份 | 来自“融合所有支线沈渊”的时间线 |
| 境界 | 真君（完整，失去自我） |
| 外貌 | 与沈渊相似，但眼神“空洞”，穿白色长袍 |
| 性格 | 冷漠、理性，认为“删除支线”是“慈悲” |
| 能力 | 叙事大道（完整），可“删除”分支 |
| 弧线 | 从“清除支线”到“保护支线”（受沈渊影响，融合斗战真君碎片后转变） |
| 结局 | 成为酆都废墟“支线社区”的守护者 |

**关键对话**：
- “我们都是被删减的坏结局，你是唯一还没被作者放弃的草稿。”
- “我宁愿‘消失’，也不愿‘变成别的什么’。”（天道的话语，被他引用）

### 3.2 换命师沈渊（交易派）

| 项目 | 内容 |
|------|------|
| 身份 | 来自“没有去纸扎城”的时间线，换命师首领 |
| 境界 | 劫仙（但存在不稳定） |
| 外貌 | 与沈渊相似，左手透明（被交易反噬） |
| 性格 | 冷静、算计，认为“命运可以交易” |
| 能力 | 命运交易（可转移任何“命运”） |
| 弧线 | 从“试图校正主角轨迹”到“被沈渊稳定存在”到“成为图书管理员” |
| 结局 | 在纸扎城文曲塔当图书管理员，平静生活 |

**关键对话**：
- “命运可以被‘交换’，但不能被‘改变’。”
- “我‘后悔’‘交易’。‘但’‘也’‘感谢’‘交易’‘让’‘我’‘遇到’‘你们’。”

### 3.3 “尸体”沈渊（因果灾）

| 项目 | 内容 |
|------|------|
| 身份 | 来自“大劫发生后的未来”，以“尸体”形态回到过去 |
| 状态 | 已死，但“意识”留在尸体中 |
| 目的 | 告诉过去的沈渊“如何避免大劫” |

**关键对话**：
- “我‘死’于‘大劫’。‘但’‘我’‘选择’回到‘过去’，‘告诉’你‘如何’避免。”

## 五、批注者（血字 & 墨字）

### 5.1 总览

批注者是“读者集体意识”的“具象化”，在《诡仙录》抄本上写批注，批注“可能”影响现实。

| 名称 | 笔迹 | 立场 | 身份猜测 | 经典批注 |
|------|------|------|----------|----------|
| 血字 | 暗红色 | 悲剧导向 | 真君沈渊的残留 / 偏好悲剧的读者 | “蠢货，你改得越早，死得越惨。” |
| 墨字 | 黑色 | 希望导向 | 文曲真君的残留 / 偏好圆满的读者 | “别理他，他是疯子。我是好人。” |

### 5.2 血字

- **性格**：毒舌、悲观、喜欢“看”角色受苦（认为“悲剧更深刻”）。
- **与沈渊互动**：早期嘲讽，中期“警告”，后期“辩论”。
- **真相**（第九卷）：是“读者意识”中“偏好悲剧”的部分。

### 5.3 墨字

- **性格**：温和、乐观、喜欢“帮助”角色（但“帮助”可能“帮倒忙”）。
- **与沈渊互动**：早期提示，中期“指导”，后期“辩论”。
- **真相**（第九卷）：是“读者意识”中“偏好希望”的部分。
