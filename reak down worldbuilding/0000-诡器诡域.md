# 《诡仙录》诡器与诡域文档

> 版本：最终定稿  
> 类型：克苏鲁式恐怖 + 东方诡仙 + 强Meta叙事 + 存在主义哲学  
> 核心精神：人以身为饵，饲诡登阶。力量即污染，改写即代价，故事从未属于你。


## 上篇：诡器

### 一、诡器总览

#### 1.1 定义

诡器，是修士在“炼诡”境将自身“记忆”、“情感”或“人格”剥离后，以诡源“固化”而成的器物。每一件诡器都是一段“被删除的人生”。

#### 1.2 诡器的本质

- **记忆武器化**：诡器不是“外物”，而是修士“自己的一部分”。使用诡器 = “调用”被删除的记忆。
- **代价守恒**：炼制诡器需要“付出”记忆/情感；使用诡器往往需要“再次”付出（额外记忆或因果）。
- **叙事残留**：诡器“记录”着被删除的叙事片段，使用诡器可能“重历”那段人生。

#### 1.3 诡器的分类

| 类型 | 描述 | 示例 |
|------|------|------|
| 记忆型 | 以特定记忆为原料，功能与记忆内容相关 | 醒世钟（母亲记忆→“清醒”） |
| 情感型 | 以情感为原料，功能与情感相关 | 忘情碑（绝情→镇压） |
| 概念型 | 以“概念”为原料（如“绝情”、“战斗”） | 哭佛像（慈悲→承载污染） |
| 叙事型 | 以“叙事片段”为原料，可改写现实 | 伪本、文曲笔 |

#### 1.4 诡器的炼制

**原料**：
- **核心材料**：一段记忆、情感或人格（必须属于炼制者自己，或通过交易获得）。
- **载体材料**：诡异之物（如镜婆婆的镜子碎片、空白《诡仙录》抄本、天魔血等）。

**步骤**：
1. 选择“原料记忆”（必须是“足够重要”的，否则诡器太弱）。
2. 用“叙事之力”将记忆“剥离”（感受剧痛，记忆从脑中“抽离”）。
3. 将记忆与载体“融合”（记忆“固化”为器物形态）。
4. 命名诡器（命名本身也是一种“叙事锁定”）。

**代价**：
- 炼制者永久失去那段记忆（不是“遗忘”，而是“从未发生过”）。
- 记忆的“删除”会“扩散”——认识记忆中人者也会逐渐“忘记”那人。


### 二、主要诡器列表

#### 2.1 醒世钟

| 项目 | 内容 |
|------|------|
| 炼制者 | 沈渊（第五卷） |
| 原料记忆 | 母亲（的所有记忆） |
| 载体材料 | 镜婆婆的镜子碎片 |
| 形态 | 银色小钟，表面有“母亲拥抱孩子”的浮雕 |
| 功能 | 敲响时，范围内所有人“清醒”看到真实的剧情线（黑线），不受幻象、诡域、叙事污染干扰 |

**经典使用场景**：
- 测试：陈舟看到自己的黑线通向“战死”。

## 下篇：诡域

### 三、诡域总览

#### 3.1 定义

诡域，是“掌诡”境修士以自身“叙事畸变”为核心，创造的“领域”。域内规则由域主制定，但必须“维持谎言”——域内居民（或诡异）必须“相信”规则，否则“设定崩塌”。

#### 3.2 诡域的本质

- **叙事气泡**：诡域是修士“叙事畸变”的“外溢”——域主的“故事”成为域内的“现实”。
- **谎言即真实**：域内规则不需要“符合”外部叙事，只需要“被信仰”。
- **域主即上帝**：域主可以在域内“改写”现实，但改写的“代价”由域主承担。

#### 3.3 诡域的等级

| 等级 | 范围 | 规则稳定性 | 示例 |
|------|------|-----------|------|
| 小型 | 数十丈 | 依赖域主“时刻”维持 | 镜婆婆的“镜中界” |
| 中型 | 数里 | 依赖“集体信念”维持 | 纸扎城（升格前） |
| 大型 | 数十里 | 规则“半独立”，可自我维持 | 酆都废墟（自然形成） |
| 世界级 | 一洲或以上 | 规则“独立”，不依赖域主 | 纸扎城（升格后的小世界） |

#### 3.4 诡域的形成

- **主动创造**：修士在“掌诡”境主动“展开”诡域（如镜婆婆）。
- **自然形成**：叙事漏洞或诡异聚集“自然”形成诡域（如酆都废墟）。
- **传承继承**：前任域主“遗留”诡域，后人“接管”（如沈渊接管纸扎城）。


### 四、主要诡域描述

#### 4.1 纸扎城

| 项目 | 内容 |
|------|------|
| 类型 | 主动创造（文曲真君）→ 接管（沈渊）→ 升格为“小世界” |
| 位置 | 纸扎洲，里界夹缝中 |
| 域主 | 文曲真君（原）→ 沈渊（现任） |
| 范围 | 整座城市（约百里） |
| 居民 | 纸人（数万）、忘川成员、妖族（后） |

**域内规则**：
- **“谎言即真实”**：只要全体居民“相信”某件事，它就会“成真”。
- **纸人特性**：纸人由纸折成，但“坚信”自己是血肉之躯。他们的“记忆”是沈渊（或前任域主）编造的“谎言”，但他们“选择”相信。
- **清明雨**：每年（时间不确定），“叙事清洗”会降临——雨水会“融化”纸人，因为“谎言被真实戳破”。升格后，清明雨变为“重绘之雨”（不会消失，只会“打湿”后重绘）。

**重要地点**：
- 文曲塔：叙事核心，存放文曲笔和文曲真君的意识碎片。
- 纸人议会：纸人们的自治机构。
- 英雄墓：埋葬陈舟、苏灵儿等忘川成员。

**历史**：
- 文曲真君创造纸扎城作为“档案馆”，保存“好故事”。
- 沈渊成为域主后，经历“清明雨”危机，选择“把自己也折成纸人”，与纸人们共同经历“死亡-重绘”。
- 纸扎城“升格”为“独立小世界”，不再只是“诡域”。

**叙事意义**：唯一一个“不被叙事完全控制”的地方——因为“集体信念”可以“改写”局部规则。

#### 4.2 酆都废墟

| 项目 | 内容 |
|------|------|
| 类型 | 自然形成（天庭崩塌后的“残骸”） |
| 位置 | 酆洲，原天庭遗址 |
| 域主 | 无（或“叙事”本身） |
| 范围 | 覆盖整个酆洲（极大） |
| 居民 | 遗蜕、被删除的角色、时间线残影 |

**域内规则**：
- **时间线叠加**：同一地点可能“同时”存在多个时间点的自己。
- **“被删除章节”堆放场**：所有被“删除”的剧情、角色、设定都“堆”在这里。
- **剧情黑洞密集**：黑洞会“吞噬”记忆和存在。

**重要地点**：
- 草稿社区：被删除的沈渊们的聚居地。
- 真君沈渊的宫殿：由“书页”堆叠而成。
- 藏书塔：收藏“被删除的剧情”的固体书。
- 斗战真君的沉睡之地。

**危险**：
- 迷失在“时间线叠加”中，分不清“哪个是现在的自己”。
- 被剧情黑洞“删除”。
- 被遗蜕或被删除角色攻击。

**叙事意义**：叙事的“废纸篓”——所有“写坏”的部分都“丢”在这里。

#### 4.3 归墟海

| 项目 | 内容 |
|------|------|
| 类型 | 自然形成（叙事“残留物”聚集） |
| 位置 | 归墟洲，里界最深处 |
| 域主 | 死亡真君（沉睡） |
| 范围 | 覆盖整个归墟洲（海洋） |
| 居民 | 遗蜕（大量）、死亡真君 |

**域内规则**：
- **重力“反转”**：越往里走，越感觉自己在“下坠”而非“前进”。
- **“没有水的海”**：海水是黑色的“被删除叙事”残留物，不是水。
- **记忆交易**：船夫（遗蜕）摆渡需要“记忆”作为船资。

**重要地点**：
- 遗蜕船夫的码头：唯一“安全”进入归墟海的入口。
- 死亡真君的沉睡之地：归墟海最深处，死亡真君的“遗蜕”漂浮于此。
- 海中心“漩涡”：通往渊界。

**危险**：
- 遗蜕攻击活人，抢夺身体。
- 叙事乱流“撕碎”意识。
- 死亡真君的“沉睡”可能被“打扰”，导致“苏醒”。

**叙事意义**：叙事的“坟墓”——被删除的存在“安息”于此。

#### 4.4 倒悬山

| 项目 | 内容 |
|------|------|
| 类型 | 自然形成（叙事“断层”） |
| 位置 | 倒悬洲，里界与渊界交界处 |
| 域主 | 无（或原初存在） |
| 范围 | 整座山（但“山顶”在渊界） |
| 居民 | 无（只有“考验”） |

**域内规则**：
- **重力“反转”**：山体倒挂，要“登山”需要“倒着走”（头朝下，脚朝上）。
- **时间“倒流”**：登山者会“看到”自己的“过去”在“倒退”。
- **三道考验**：记忆关、情感关、自我关。

**重要地点**：
- 作者碑：山顶的一块石碑，据说刻着《诡仙录》的“原初第一行字”。
- 原初存在的“核心”：山顶的“光”，可以与“作者”对话。

**危险**：
- 无法通过考验者“迷失”在时间倒流中，变成“空白”。
- 直视原初存在可能“疯掉”。

**叙事意义**：叙事的“门槛”——唯一能“直视”渊界而不立即死的地方。

#### 4.5 镜中界（镜婆婆的诡域）

| 项目 | 内容 |
|------|------|
| 类型 | 主动创造 |
| 域主 | 镜婆婆（已“转化”） |
| 范围 | 约数十丈（可扩展） |
| 规则 | 反射一切攻击；复制域内存在（制造镜像分身） |
| 弱点 | 无法“创造”，只能“反射”；“盲视”可免疫 |
| 结局 | 镜婆婆被沈渊“改写记忆”后，镜中界“消失” |

#### 4.6 血海（血屠的诡域）

| 项目 | 内容 |
|------|------|
| 类型 | 主动创造 |
| 域主 | 血屠（已死） |
| 范围 | 约百丈 |
| 规则 | 血海“淹没”一切，污染“信念”（纸人被血海浸泡会“褪色”） |
| 弱点 | 净水（或“信念”制造的净水）可“冲走”血海 |
| 结局 | 血屠被纸鸢束缚后自爆，血海“消散” |

#### 4.7 万病缠身（秦无病的诡域）

| 项目 | 内容 |
|------|------|
| 类型 | 主动创造 |
| 域主 | 秦无病（后“疯掉”） |
| 范围 | 约数百丈 |
| 规则 | 范围内的敌人会“患病”，不是肉体的病，而是“叙事病”（如“记忆疟疾”、“因果癌”） |
| 弱点 | “集体信念”可“治愈”；域主本人也会被“反噬” |
| 结局 | 秦无病被沈渊“反噬”，患上“因果癌”，变成“疯子” |

#### 4.8 纸界（沈渊的诡域，纸人形态）

| 项目 | 内容 |
|------|------|
| 类型 | 主动创造（作为纸人域主） |
| 域主 | 沈渊（纸人形态） |
| 范围 | 约百丈（可扩展） |
| 规则 | 将周围的一切“纸化”——敌人的武器变成纸、身体变成纸（防御力下降），忘川成员的纸化武器变得更强 |
| 弱点 | 怕火（尤其是“叙事之火”）；怕“怀疑”（敌人“不相信”自己是纸，规则减弱） |
| 使用场景 | 纸扎城保卫战（对魔宗） |

#### 4.9 梦境社区（沈渊的“内部”诡域）

| 项目 | 内容 |
|------|------|
| 类型 | 自然形成（沈渊的“梦境”） |
| 域主 | 沈渊（但“住”着其他沈渊） |
| 范围 | 沈渊的“意识”内 |
| 居民 | 独臂沈渊、老沈渊、疯沈渊、纸鸢（投影，后离开） |
| 规则 | 梦中的“沈渊们”可以“投影”到现实（短暂战斗）；沈渊可以在梦中“访问”他人的记忆 |
| 危险 | 可能“迷失”在多重自我中，分不清“谁是主”；外人进入可能“被困” |
| 结局 | 纸鸢“离开”后，社区“平静”，成为沈渊的“内心花园” |


### 五、其他特殊物品/概念

#### 5.1 《诡仙录》（真本）

| 项目 | 内容 |
|------|------|
| 本质 | 叙事的“总纲”，原初存在的“梦话” |
| 形态 | 一本无头无尾的古书，页面会自动“更新” |
| 功能 | 记录“正在发生”的故事；可以“划掉”改写（需代价）；可以“续写”（需权限） |
| 限制 | 无法“删除”已经“写定”的内容（只能“划掉”，代价极高） |
| 持有者 | 沈渊（第一卷发现，后一直携带） |
| 叙事意义 | 既是“预言书”，也是“记录书”。沈渊最终在书上“续写”了“永恒黄昏”。 |

#### 5.2 纸棺

| 项目 | 内容 |
|------|------|
| 炼制者 | 沈渊（第五卷） |
| 材料 | 纸人们的“纸张”（宣纸） + 天魔血 |
| 形态 | 黑色纸棺 |
| 功能 | 沈渊“诡仙”境沉睡的“容器”，意识“下沉”到叙事深层 |
| 特殊 | 纸棺与沈渊的“纸人”身份契合，沉睡时间流速与外界“1:100” |
| 结局 | 沈渊苏醒后，纸棺“保存”在文曲塔（备用） |

#### 5.3 龙血

| 项目 | 内容 |
|------|------|
| 来源 | 妖庭“龙尸”的血液（由白媚提供） |
| 功能 | 强化纸扎城的“信念”之力；纸人“浸泡”后可获得“临时”能力（如飞行） |
| 代价 | 带有“死亡”叙事污染，纸扎城天空偶尔出现“龙影” |
| 使用 | 纸鸢转化仪式中用了一部分；剩余“保存”在文曲塔 |

#### 5.4 真人之血

| 项目 | 内容 |
|------|------|
| 来源 | 归墟海遗蜕（一位“劫仙”遗蜕） |
| 功能 | 纸鸢“转化”为真人所需的核心材料 |
| 代价 | 沈渊付出一段“纸扎城升格记忆”（代价由沈渊承担） |
| 使用 | 纸鸢转化仪式中使用 |

#### 5.5 天魔血

| 项目 | 内容 |
|------|------|
| 来源 | 魔宗宗主“天魔”的血液（魔宗“赠送”） |
| 功能 | 可“抵抗”叙事污染；滴在纸棺上使纸棺变为“黑色” |
| 代价 | 可能“吸引”天魔的注意（但天魔已消散） |
| 使用 | 炼制纸棺时使用；剩余“保存”在文曲塔 |


## 文档结束

> **最后的话**：
>
> 诡器是“被删除的记忆”，诡域是“被改写的规则”。它们都是“叙事”的“具象化”——修士通过“付出”自己的一部分，来“获得”改变“故事”的能力。沈渊的“醒世钟”让他“清醒”，但也让他“忘记”；纸扎城的“谎言即真实”让他“守护”，但也让他“失去自我”。这就是诡仙的“交易”——没有“免费”的力量，只有“等价”的代价。
