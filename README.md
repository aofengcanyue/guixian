# 诡仙录

百万字级长篇网文项目。

## 当前大纲版本

- `docs/大纲-v0.2-百万字长篇节奏版.md`

## 核心定位

《诡仙录》定位为：**诡异修仙求生文，后期转黑暗救世**。

前期不是救世，不是推翻世界，而是：

> 主角沈照在仙门中活下来，并救下第一个不该被炼成丹的人。

中期逐步扩大为：

> 从救一个人，到救一群人，再到建立不依赖仙门的生存方式。

后期进入：

> 追查仙道病灶、揭开飞升骗局、重定人间与仙道的关系。

## 写作总原则

1. 百万字级长篇节奏，真相分卷揭露。
2. 主角循序渐进成长，前期弱、能忍、会装、靠观察求生。
3. 仙门必须有画皮，不能一开始写成明面魔门。
4. 《诡仙录》不是万能系统，观病、录名、剥病都有门槛和代价。
5. 主题落点在人：人可以死，但不能被当成丹、当成矿、当成香火、当成仙路的柴。
